VideoSense Agent

Ask your video library in plain English — it watches, reasons, and answers, with the clip and the chart to prove it.

  • Conversational agent over a personal video library: natural-language Q&A answered with playable clips and charts as evidence.
  • Gemini 2.5 multimodal reads the actual video content, not just metadata.
  • Agent loop with tool use: SQL over extracted video facts, semantic search (PostgreSQL + pgvector), inline clip playback, chart rendering, web search.
  • Production-grade: FastAPI backend on Google Cloud Run, per-request token/cost telemetry piped to BigQuery, cross-session user memory, 146 passing tests.

DINOv2-GraphNAV

Visual navigation via an offline DINOv2 feature graph and online global matching, built for an Embodied AI Visual Navigation Challenge.

  • Offline: DINOv2-B/14 global embeddings + PCA-reduced local patch tokens build a weighted K-NN topological graph from exploration trajectories.
  • Online: real-time localization with a lightweight DINOv2-S/14 backbone and FAISS nearest-neighbor search; A*/Dijkstra routing with multi-step next-best-view guidance.
  • PyTorch
  • DINOv2
  • FAISS
  • Graph Search
